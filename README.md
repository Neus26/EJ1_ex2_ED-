# Clases
Este proyecto cuenta con diferentes clases para su posible ejecución, las cuales tienen los nombres en frances e ingles:

Main.java - clase principal
Design.java - diseño
Film.java - clases de peliculas
Realisateur.java - los directores
Ajouter_Film.java - para añadir peliculas
Supprimer_Film.java - para borrar peliculas
Vente_Film.java - para vender peliculas
Film_Action.java - peliculas de accion
Film_Romance.java - peliculas de romance
Film_ScienceFiction.java - peliculas de ciencia ficcion
